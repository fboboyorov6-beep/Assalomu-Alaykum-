# Assalomu alaykum

Uzbek-first Muslim daily companion website prototype.

## Features

- Qibla compass modal
- Prayer times by selected Uzbekistan region
- Nearby masjid and Hijri calendar entry cards
- Uzbek-first visual style with Islamic green and gold accents

## Run Locally

```bash
python3 -m http.server 4174
```

Then open:

```txt
http://127.0.0.1:4174/
```

## Deploy

This is a static website. Upload these files to any static hosting provider:

- `index.html`
- `styles.css`
- `script.js`
- `assets/`
- `site.webmanifest`
- `robots.txt`
